Több operációs rendszer telepítése, eltávolítása
XP / Vista / 7 / Linux, a sorrend is ilyen telepítésnél

Windows-újratelepítés Linux mellett
avagy a Grub helyreállítása
  1. Bebootolsz LiveCD-rõl (pl. Ubuntu)

  2. Nyitsz egy terminált

  3. Beírod ezt:
    KÓD
    sudo grub

    Ezután a terminálban grub> -ot fogsz látni

  4. KÓD
    find /boot/grub/stage1

    Ez megadja, hogy melyik partíción van a Grub (ami nem tûnt el, csak megsérült a Windows telepítése során).

  5. KÓD
    root (hd?,?)

    Értelemszerûen a két kérdõjel helyére az kerül, amit az elõbb kidobott a terminál.

  6. KÓD
    setup (hd?)


  7. KÓD
    quit
Újraindítva elvileg újra a Grub fogad.

Ez a leírás még a régi Grub-hoz készült. Útmutató a Grub2-höz az Ubuntu http://ubuntu.hu/node/13596

Windows XP/Vista/7 eltávolítása XP mellõl

Windows Vista/7 eltávolítása
  1. Bebootolsz az XP telepítõ CD-rõl
  2. Javítókonzolt választasz
  3. Kiadod a fixboot és a fixmbr parancsokat!
Windows XP eltávolítása a másik XP mellõl
  1. Start --> Vezérlõpult --> Rendszer --> Speciális --> A harmadik Beállítások gomb --> Szerkesztés
    Itt nagyon körültekintõen távolítsuk el azt a Windowst, amire nincs szükség. A partíciók számozásából kiderül, hogy melyik melyik.

  2. Ezt követõen Sajátgép jobb katt --> Kezelés --> Lemezkezelés
    Formázd le vagy töröld le a másik partíciót.
Ezután a lemezkezelõvel simán tudod formázni azt a partíciót, ami a törölni kívánt rendszert tartalmazza.

Windows XP (újra)telepítése Vista/7 mellé

Telepítsük fel az XP-t a kiszemelt partícióra.

Az XP nem ismeri fel a nála újabb rendszereket, ezért felülírja a Vista/7 indítóját. Semmi pánik, bootoljunk be a Vista/7 telepítõlemezérõl, majd válasszuk a javítást. Jó eséllyel automatikusan megoldja, ha nem, akkor indítsuk el a parancssort és gépeljük be az alábbi parancsokat, minden sor után entert ütve:

KÓD
bootrec.exe /fixboot
bootrec.exe /fixmbr


Indítsuk újra a gépet, a Vistának/7-nek kell indulnia. Töltsük le az http://neosmart.net/dl.php?id=1 nevû programot. Az Add/Remove Entries alatt válasszuk a Windows NT/2k/XP/2k3-az a legördülõ menübõl, adjunk meg egy nevet (mondjuk Windows XP), ezután Add Entry. Partíciónak az XP-ét adjuk meg. Save. Ezután a Manage Bootloader résznél Reinstall the Vista Bootloader legyen kiválasztva és nyomjuk meg a Write MBR gombot.

Indítsuk újra a gépet, egy indítómenünek kell fogadnia, amiben választhatunk a két operációs rendszer közül.

hogyan lehetne megoldani a ClearType tuner és a TweakUI telepítését HFSLIP-pel?

Én nem a HFSLIP integrálását használom erre, hanem van egy install.cmd batch fájl az $OEM$ mappában.

KÓD

@echo off

TITLE Windows XP SP3 - Unattended Installation

ECHO Over the next few minutes you will see automated installations

start /wait %systemdrive%\Apps\PowerToys\ClearTypeTuner.exe /s /v/qn

ECHO.
ECHO Installing PowerCalc
ECHO Please wait...
start /wait %systemdrive%\Apps\PowerToys\PowerCalc.exe /s /v/qn

ECHO.
ECHO Installing TweakUI
ECHO Please wait...
start /wait %systemdrive%\Apps\PowerToys\TweakUI.msi /qb

ECHO.
ECHO Installing PowerToys Hungarian Language Skin
ECHO Please wait...
start /wait %systemdrive%\Apps\PowerToys\PowerToysLanguageSkin_HU.msi /qn


EXIT

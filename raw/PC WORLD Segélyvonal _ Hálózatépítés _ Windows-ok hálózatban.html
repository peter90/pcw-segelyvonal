Windows 7 Otthoni csoport
A new hope

A Microsoft ezzel az új eszközzel szeretné megkönnyíteni a fájlok megosztását otthonainkban, ami szép gesztus, hátulütõje, hogy csak Windows 7 alatt érhetõ el. A demóban egy Home Premium és egy Professional 7 vesz részt, alapfelállásban a Home Premium rendelkezik egy otthoni csoporttal, a Professional pedig csatlakozhat hozzá, de nézzük, hogy kezdõdik ez az egész.

Otthoni csoport létrehozása

  1. Az egész a Windows telepítésekor kezdõdik, mégpedig a hálózat helyének kiválasztása után, minden automatikus, nekünk csak a "Könyvtárak" megosztását van lehetõségünk beállítani (nem minden esetben).
  2. Telepítés után az Otthoni csoportot a Vezérlõpult --> Hálózati és Megosztási központban találjuk meg, nem meglepõ módon Otthoni csoport néven.
    http://img97.imageshack.us/i/94162889.png/
  3. Lépjünk is bele, lássuk, mi vár ránk!
    http://img51.imageshack.us/i/26314450.png/

    Látható, hogy egybõl reklamál, hogy nincs megosztva semmink se, de legalább csatlakoztunk egy Otthoni csoportba, ami világos, ez a sajátunk, ami automatikusan létrejött.
  4. Válasszuk ki, mit szeretnénk megosztani!
    http://img191.imageshack.us/i/59473919.png/
  5. Továbbra kattintva generál nekünk egy kulcsot, amivel lehetõségünk van más Windows 7-es gépekrõl csatlakozni ebbe a csoportba. A kulcsot valóban érdemes lementeni, annak ellenére, hogy lehetõségünk lesz elõbányászni.
    http://img526.imageshack.us/i/49220087.png/
  6. Befejezés gombra kattintva valóban befejezõdik itteni tevékenységünk.
    http://img27.imageshack.us/i/32637415.png/

    Érdemes fejben tartani a bekarikázott részt. Mi is ez? Igen, a jelszó megtekintése.
Csatlakozás otthoni hálózatba
  1. Ugyanott kezdõdik a dolog, csak más képernyõ fogad:
    http://img694.imageshack.us/i/31878637.png/

    Windows 7 Professional felõl látjuk a dolgot, észlelte a hálózatban meglévõ Otthoni csoportot, csatlakozást ajánl, ami nekünk nagyon jó.
  2. Csatlakozás gombra kattintva ugyanaz a "Könyvátrak" megosztása fogad, majd a jelszót kell beírni.
    http://img705.imageshack.us/i/13779671.png/
  3. A Tovább gombra kattintva kis szöszmötölés után értesít, hogy csatlakozott az Otthoni csoportba.

  • Lehetõség van beállítani, hogy fiókok szerint menedzselõdjön ez az egész, de bízzuk ezt inkább a Windows-ra. (Ahhoz hasonlít, amikor otthon rejtett felhasználó segítségével teremtünk megosztást.)

Távoli asztali kapcsolat
A leírás alapvetõen Windows XP-hez van, de aki képes azon megcsinálni, annak nem lesz gondja se Vista-n, se 7-en

http://www.pcworld.hu/forum/index.php?showtopic=13587


Ahova kapcsolódunk
  1. Szükség van egy felhasználóra, akinek van jelszava (nélküle nem megy). Start --> Beállítások --> Vezérlõpult --> Felhasználói fiókok
    Sokféle megoldás van, használtam már 1 felhasználói fiókot is rá, de akár lehet kettõt is: automata bejelentkezésen van az egyik, míg a másik csak Távoli asztali kapcsolatnál van használatban.
  2. Ez még önmagában kevés, engednünk kell a Távoli asztalos kapcsolódást: Start --> Beállítások --> Vezérlõpult --> Rendszer --> Távoli használat fül:
    1. Pipáljuk be a Távoli Asztalt;
    2. Kattintsunk a felhasználók kijelölésére;
    3. Hozzáadás;
    4. Beírjuk a nevet, majd a névellenõrzésre nyomunk;
    5. Mindenre OK;
  3. Ha használunk tûzfalat, akkor ki kell engednünk a Távoli asztal szolgáltatást és/vagy a 3389-es portot.

    Ha router mögül internetezünk, akkor nyitnunk kell egy portot virtual server/port forwarding menüpont alatt.
Ahonnan kapcsolódunk
Start --> Programok --> Kellékek --> Távoli asztali kapcsolat
  • Figyeljünk a portra, és igen, külsõ IP kell;
  • A beállítások gombra kattintva megadhatunk felhasználót, megmondhatjuk, mi kerüljön rá a "erre a gépre", meg még néhány más dolgot is beállíthatunk;

Figyelem
Távoli asztali kapcsolódáskor a távoli gép felhasználója értesítést kap csatlakozási szándékunkról, õ dönti el, hogy átadja-e a rendszer irányítás. Egyszerre 1 ember lehet a gép ura Távoli asztali kapcsolattal.

Hogyan kell a Távoli asztali elérésbõl kitörölni a már nem használatos IP címeket?
HKEY_CURRENT_USER\Software\Microsoft\Terminal Server Client\Default
HKEY_CURRENT_USER\Software\Microsoft\Terminal Server Client\Servers

illetve

HKEY_USERS\...\Software\Microsoft\Terminal Server Client\Default
HKEY_USERS\...\Software\Microsoft\Terminal Server Client\Servers

Összeférhetõség
Windows XP, Windows Vista, Windows 7 együttmûködik, Windows 7-ben kellhet lelõni a Távoli asztali kapcsolat új hitelesítését.

  • WinVnc
  • RealVNC
  • UltraVNC
  • LogMein
  • TeamViewer

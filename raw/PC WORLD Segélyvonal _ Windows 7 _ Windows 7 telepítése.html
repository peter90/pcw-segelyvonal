"Cannot boot from CD - Code: 5" hibaüzenet jelenik meg a telepítésnél
Megoldás (32 bites 7-hez 32-bites Vista, 64 a 64-eshez):
  1. Vista ISO megnyitása az UltraISO 9.3.0.2612 programmal, jobb felsõ ablakban kitörölni mindent (lemez tartalma majdnem).
  2. VirtualCloneDrive-val bemountolni a Windows 7 ISO-t.
  3. Az UltraISO-ban rátallózni a mountolt Windows 7 lemezre, és a teljes telepítõ tartalmát átmásolni az egérrel a jobb felsõ ablakba a törölt Vista fájlok helyére.
  4. Mentés másként, dolgozik pár percet, és végül írhatjuk ki a telepítõnket.
